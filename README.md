# 📱 Contact Manager (Jira Project)

## 📌 Description
Contact Manager est une application Python de gestion de contacts développée dans un cadre académique.  
Le projet évolue en plusieurs versions et inclut une interface graphique ainsi qu’une architecture orientée objet.

---

## 🎯 Objectif du projet
Ce projet a été réalisé afin de :
- Appliquer la programmation orientée objet (OOP)
- Gérer des données utilisateurs (contacts)
- Manipuler des fichiers en Python
- Développer une interface graphique (GUI)
- Utiliser Git et GitHub pour le versioning

---

## 🚀 Fonctionnalités

- ➕ Ajouter un contact
- ✏️ Modifier un contact
- ❌ Supprimer un contact
- 🔍 Rechercher un contact
- 📄 Afficher la liste des contacts
- 💾 Sauvegarde des données dans un fichier texte
- 🖥️ Interface graphique (Tkinter dans la version v0.0.3)
